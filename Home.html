Let the games begin!

Game objective

The goal of this exercise is to get more practice building websites and getting familiar with Git and GitHub. As a team, we will be trying to make this website as fun and random as possible, with everyone working on it at the same time (or the same work, since we may be split across shifts). Some specific tasks to try and do during time might include:

  • Create branches
  • Create pull requests
  • Generate merge conflicts and resolve them
  • Get practice editing the readme and using markdown
  • Add images and embed content onto the site
  • Style with CSS
  • Add more web pages, css files, etc.
  • Create issues on GitHub and request reviewers

Rules

As much as possible, try to communicate with each other as a team. Make comments on PRs, request each other as reviewers, approve pages, get to know each other. You can post your hobbies or pictures, embed videos, etc.

Other rules:

  • Don't delete the entire project or mess with the permissions
  • Respect everyone else
  • No offensive pictures or other content (keep it work-appropriate)
